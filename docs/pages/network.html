Client API

Network

Client-side request module. Supports callback-based and Promise-based async patterns, automatic retry on timeout, and per-request cancel.

local network = RoExpress.GetNetwork()   -- typed
-- or: RoExpress("Network")                -- returns any

Callback form

All methods return a requestId that can be passed to network:Cancel().

GET network:Get(route, data?, callback) → requestId
POST network:Post(route, data?, callback) → requestId
PUT network:Put(route, data?, callback) → requestId
DELETE network:Delete(route, data?, callback) → requestId
network:Get("player/123", nil, function(res)
    if res.type == "success" then
        print(res.data)
    else
        warn(res.message)
    end
end)

network:Post("player/123/update", { name = "NewName" }, function(res)
    print(res.status, res.data)
end)

Blocking form

Yields the current thread until a response is received. Returns both the requestId and the response. Must be called inside a coroutine or task.spawn.

GET network:GetSync(route, data?) → requestId, NetworkResponse
POST network:PostSync(route, data?) → requestId, NetworkResponse
PUT network:PutSync(route, data?) → requestId, NetworkResponse
DELETE network:DeleteSync(route, data?) → requestId, NetworkResponse

Promise form

Chainable async API. See Promise module for full docs.

GET network:GetAsync(route, data?) → Promise
POST network:PostAsync(route, data?) → Promise
PUT network:PutAsync(route, data?) → Promise
DELETE network:DeleteAsync(route, data?) → Promise
network:GetAsync("inventory/items")
    :Then(function(res) return res.data.items end)
    :Then(function(items) populateUI(items) end)
    :Catch(function(err) warn(err.message) end)

Parameters

ParameterTypeDescription
routestringRoute string, must match a registered server handler (e.g. "player/123")
dataany?Optional payload to send. POST requires a body; PUT expects one. GET/DELETE accept optional data.
callbackfunctionReceives NetworkResponse. Called once when the server responds or the request times out.

NetworkResponse

FieldTypeDescription
type"success" | "error" | "timeout"Result classification
statusnumber?HTTP-style status code set by res:Status()
dataany?Response data from res:Send(data)
messagestring?Error message from res:Error(message) or built-in middleware rejections
compressedboolean?true if the response was Deflate-compressed and auto-decompressed

Cancel

network:Cancel(requestId)

Cancels a pending request. The callback will not be called. Has no effect if the response has already arrived.

local id = network:Get("slow/resource", nil, function(res) print(res.data) end)

-- Cancel it if the player leaves before it resolves
Players.PlayerRemoving:Connect(function()
    network:Cancel(id)
end)

Retry

Network automatically retries timed-out requests up to the configured maximum. The default is 2 retries with exponential backoff. Configure on the RoExpress root:

-- Not yet configurable per-request — global setting on module init
-- Retries: 2 (default), Timeout: 10s (default)

Destroy

network:Destroy()

Disconnects the RemoteEvent listener and cancels all pending requests. Call on cleanup if needed; normally not required.
