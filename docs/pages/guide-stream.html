FPS with Stream

Stream sends player state to all clients every tick — one packed buffer, one FireAllClients. This guide walks through a typical FPS setup with lag compensation for hit validation.

Stream uses generic channels. The steps below use a single players channel. You can add more channels for bullets, vehicles, or any other high-frequency state — see Stream.

Server setup

local Stream = RoExpress("Stream")

-- define schema: fields broadcast every tick
local ch = Stream.Channel("players", {
    x       = Stream.Types.Float32,
    y       = Stream.Types.Float32,
    z       = Stream.Types.Float32,
    rotY    = Stream.Types.Float16,
    health  = Stream.Types.UInt8,
})

game:GetService("RunService").Heartbeat:Connect(function()
    for _, player in ipairs(game.Players:GetPlayers()) do
        local char = player.Character
        if not (char and char:FindFirstChild("HumanoidRootPart")) then continue end
        local cf = char.HumanoidRootPart.CFrame
        ch:Set(player.UserId, {
            x = cf.X, y = cf.Y, z = cf.Z,
            rotY = cf.LookVector.X,
            health = char.Humanoid.Health,
        })
    end
    ch:Flush()   -- one FireAllClients with all records
end)

Client subscription

local Stream = RoExpress("Stream")

Stream:Subscribe("players", function(userId, state)
    local indicator = NameplateService:Get(userId)
    if indicator then
        indicator.Position = Vector3.new(state.x, state.y, state.z)
        indicator.Health   = state.health
    end
end)

Lag compensation

Stream records a history buffer server-side. When a hit claim arrives, look back to the timestamp the client was at when they fired.

app:Post("combat/hit", function(Player, Payload, req, res)
    local ts     = req.data.timestamp  -- client clock at fire
    local target = req.data.targetId

    -- rewind channel to when client fired
    local snapshot = ch:Rewind(target, ts)
    if not snapshot then res:Status(400):Error("No snapshot"); return end

    local hitPos = Vector3.new(snapshot.x, snapshot.y, snapshot.z)
    local claimPos = req.data.hitPosition
    if (hitPos - claimPos).Magnitude > 8 then
        res:Status(400):Error("Position mismatch"); return
    end

    -- valid hit — apply damage
    DamageService:Apply(target, req.data.damage)
    res:Send({ confirmed = true })
end)
