Base64

Encode and decode utility for binary-safe string transport. Used internally by Stream and Codec. Available as a standalone utility if you need to shuttle binary data through a route response.

local Base64 = RoExpress("Base64")

API

Base64:Encode(data: string) → string
Base64:Decode(data: string) → string
Base64:EncodeTable(tbl: {}) → string
Base64:DecodeTable(data: string) → {}

Usage

local Base64 = RoExpress("Base64")

-- Encode raw binary string
local encoded = Base64:Encode(binaryString)
local decoded = Base64:Decode(encoded)

-- Encode/decode a table (JSONEncodes then Base64s)
local str  = Base64:EncodeTable({ items = { 1, 2, 3 } })
local tbl  = Base64:DecodeTable(str)
print(tbl.items)  -- { 1, 2, 3 }

When to use it

You rarely need Base64 directly. Codec handles compression + encoding internally, and Stream handles all binary serialisation. Use Base64 manually only if you need to transport raw binary data through a standard res:Send() response or store it as a string in DataStore.
